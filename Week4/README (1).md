### ขั้นตอนที่ 6: คำนวณดัชนีเชิงสเปกตรัม

ใช้ฟังก์ชันเพื่อป้องกันการหารด้วยศูนย์

```python
def safe_index(a, b):
    denom = a + b
    out = np.where(denom == 0, np.nan, (a - b) / denom)
    return out.astype("float32")

def safe_divide(num, denom):
    out = np.where(denom == 0, np.nan, num / denom)
    return out.astype("float32")
```

ดัชนีที่ใช้ได้แก่

```python
ndvi  = safe_index(nir, red)
ndwi  = safe_index(green, nir)
ndbi  = safe_index(swir, nir)
gndvi = safe_index(nir, green)
ndmi  = safe_index(nir, swir)
nbr   = safe_index(nir, swir)
evi   = safe_divide(2.5 * (nir - red), (nir + 6 * red - 7.5 * blue + 1))
```
